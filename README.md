# Smart Face Recognition Attendance System

## Overview
A real-time face recognition–based attendance system using Python and OpenCV.
The system detects and recognizes faces, then automatically logs attendance into an Excel (CSV) file.

## Features
- Face detection and recognition using LBPH algorithm
- Automated attendance logging
- Excel/CSV report generation
- Image-based and webcam-supported architecture
- Google Colab compatible implementation

## Technologies Used
Python, OpenCV, LBPH, Pandas, Flask, HTML, CSS, JavaScript

## Dataset Structure
dataset/
 ├── 0_PersonName/
 ├── 1_PersonName/

(Note: Full dataset not uploaded due to size and privacy constraints. Sample images provided.)

## How to Run
1. Install dependencies  
   `pip install -r requirements.txt`
2. Run the notebook in Google Colab or locally

## Results
- Achieved over **90% recognition accuracy**
- Tested on 1000+ face images
